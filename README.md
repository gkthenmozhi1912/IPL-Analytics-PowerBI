# IPL-Analytics-PowerBI
The IPL Analysis Dashboard (2008–2025) is an interactive Power BI dashboard developed to analyze the Indian Premier League across multiple seasons. The dashboard provides insights into tournament winners, team performance, player achievements, scoring statistics, and points table standings through dynamic visualizations and filters.

## Project Objective
Analyze IPL match data to identify performance trends.
Compare team performances across different seasons.
Evaluate player contributions through batting and bowling statistics.
Visualize key performance indicators using interactive dashboards.
Support data-driven decision-making with meaningful insights.

## Tools & Technologies
Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel (Dataset Source)

## Key Insights
Identified the most successful IPL teams based on total matches won.
Compared team performance across multiple seasons.
Analyzed player performance using batting and bowling statistics.
Evaluated match-winning trends across different venues.
Built an interactive dashboard to simplify IPL data exploration.
